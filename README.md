#### 项目介绍
 这是一个基于`http://music.163.com/api/search/pc/` 接口的vue pc端播放器的小demo

### 启动

1. 使用npm 或者 yarn

```js
npm install
yarn 
```

2.使用yarn或npm启动
```js
npm run serve
yarn serve
```
